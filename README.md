# Telegram File Converter Bot

A versatile Telegram bot that handles fast, in-chat file conversions for images, media, and documents.

## Features
- **Images:** Convert between JPG, PNG, JPEG, and WEBP.
- **Video & Animation:** Convert MP4, AVI, MKV, and MOV to GIF, or GIF to MP4.
- **Audio:** Convert MP3, WAV, OGG, and M4A, including formatting as Telegram Voice Notes.
- **Documents:** Convert DOCX to PDF or TXT, and PDF to DOCX.

## Quick Start
1. Clone this repository to your local machine.
2. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
3. Paste your telegram bot token from @BotFather
4. Run the program
   ```bash
   python main.py
